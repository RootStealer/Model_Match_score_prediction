## Using Linear Regression
Linear regression is a fundamental statistical and machine learning technique used to model the relationship between a dependent variable (the outcome you want to predict) and one or more independent variables (the factors influencing that outcome). It works by fitting a "line of best fit" through data points to minimize the difference between actual and predicted values.
* Simple Linear Regression: Uses exactly one independent variable to predict a single outcome (e.g., predicting weight based only on height).
* Multiple Linear Regression: Uses two or more independent variables to predict an outcome (e.g., predicting house prices using square footage, number of rooms, and location).
* Best-Fit Line: Also called the regression line, this is the straight line that most accurately represents the data pattern.
* Residuals: These are the "errors" or differences between the observed data points and the values predicted by the model.
